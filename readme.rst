###################
Aplikasi Enkripsi AES-128
###################

Aplikasi ini digunakan untuk mengamankan data berupa
file/text menggunakan algoritma AES-128.
Aplikasi ini di buat dengan pemrograman PHP menggunakan Framework Codeigniter
